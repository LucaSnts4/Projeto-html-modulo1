Internet, origem -

Em 1960 surgiu um novo meio de comunicação para ajudar pesquisadores governamentais da época, a internet. Os primeiros computadores dos anos 60 eram grandes demais, dificultando a sua mobilidade e para fazer uso de qualquer informação armazenada no computador era necessário ir até ele, tendo que fazer até se possível grandes viagens. O que acelerou essa evolução da internet foi o avanço da Guerra Fria. Naquela época todos queriam estar na frente em algo, e como um contra-ataque para troca mais rápida de informações os EUA criou uma rede chamada ARPANET que conseguiu os primeiros feitos com relação à internet, ligando universidades e ajudando cientistas para troca de informações e recursos. Até então as redes não tinham uma forma padrão, até que um novo protocolo foi criado, o TCP/IP, permitindo que outros computadores em redes diferentes pudessem se comunicar, então a ARPANET decidiu adotar esse novo padrão geral então criando a internet, onde todas as redes pudessem “conversar”.

Client e server, e sua ligação -

Os clients são o lado que solicita algo, com a ajuda da conexão da internet, fazendo com que exista o acesso a aplicações web e outros serviços.

Os servidores são computadores que têm o papel de realizar e finalizar a solicitação do client, onde dentro do servidor será guardado todo tipo de página, site ou aplicativo.

Quando o client entra no seu PC e usa um browser para fazer uma certa pesquisa, essa pesquisa vai para o server onde ele vai buscar a informação do documento daquela pesquisa e enviar uma cópia para o client, onde vai ficar armazenado como um Cache, para caso o client acessar de novo aquela página ou o que seja, não precise baixar em toda solicitação que acontecer.

HTML, o começo da web -

A primeira versão do HTML foi escrita por Tim Berners-Lee em 1993, e com o passar do tempo foram criadas várias versões do HTML e atualmente estamos na versão HTML5. Ao que se confunde, o HTML não é uma linguagem de programação, como o seu nome mesmo diz, HyperText Markup Language, é uma linguagem de marcação para criação de páginas web, por meio de marcadores que são as tags e seus atributos, nela vai ser ditado como tudo vai ser apresentado, montando assim o corpo da página, se vai ter texto, imagem, vídeo ou outro tipo de conteúdo.

Exemplos de Tags HTML -

As tags HTML são usadas para informar ao navegador como vai ser a estrutura do site, com elas é possível montar um site de informação, uma rede social, um site para formulário e muitas outras opções de aplicação web, veja alguns exemplos de tag abaixo:

Tag "p" - serve para colocar um parágrafo.

Tag "ul" - define uma lista não ordenada.
Tag "ol" - define uma lista ordenada.

Tag "h1" ao "h6" - São usados para títulos, como uma hierarquia, sendo o h1 maior e vai diminuindo até o h6.
